Задача: Создание блога

Контекст:
Вам необходимо создать простое веб-приложение для блога, где пользователи могут создавать, просматривать, редактировать и удалять записи.

Этапы работы:

1. Настройка проекта:
   - Создайте новый проект Django с помощью команды django-admin startproject blog_project.
   - Создайте новое приложение внутри проекта с помощью команды python manage.py startapp blog.

2. Создание модели:
   - Определите модель Post в файле models.py приложения blog с полями:
     - title (CharField)
     - content (TextField)
     - created_at (DateTimeField, автоматически устанавливаемый на текущее время)

3. Миграции:
   - Выполните команды для создания и применения миграций:
     
     python manage.py makemigrations
     python manage.py migrate
     








4. Создание представлений (views):
   - Реализуйте следующие представления:
     - post_list: отображает список всех записей блога.
     - post_detail: отображает детали одной записи по её ID.
     - post_create: форма для создания новой записи.
     - post_edit: форма для редактирования существующей записи.
     - post_delete: подтверждение удаления записи.

5. Настройка URL-адресов:
   - В файле urls.py приложения blog настройте маршруты для всех созданных представлений.

6. Создание шаблонов:
   - Создайте HTML-шаблоны для каждого представления:
     - post_list.html: отображает список записей с ссылками на детали.
     - post_detail.html: отображает заголовок и содержимое записи.
     - post_form.html: форма для создания и редактирования записи.
     - post_confirm_delete.html: подтверждение удаления записи.

7. Тестирование:
   - Запустите сервер разработки с помощью команды python manage.py runserver.
   - Проверьте функциональность приложения в браузере.

Ожидаемые результаты:

- Студенты должны представить работающий блог с возможностью добавления, редактирования, удаления и просмотра записей.
- Обсуждение возможных улучшений, таких как добавление авторизации пользователей или комментариев к записям.

Эта задача поможет студентам освоить основные концепции Django, такие как модели, представления, шаблоны и маршрутизация.
